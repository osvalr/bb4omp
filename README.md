## Bitbucket flavor for OmniMarkupPreviewer SublimeText plugin

This is a theme taken from Bitbucket css and adapted to be used within [OmniMarkupPreviewer](https://github.com/timonwong/OmniMarkupPreviewer) which is a Sublime Text 2&3 plugin.

## Installation
 1. Merge the contents of [OmniMarkupPreviewer](OmniMarkupPreviewer/) directory to `~/.config/sublime-text-X/Packages/OmniMarkupPreviewer` (replace `X` with the version of Sublime Text you're using)


## Usage

 1. Go to `Preferences` > `Package Settings` > `OmniMarkupPreviewer` > `Settings-Default`
 2. Find the following line `"html_template_name": "github",` and change it by `"html_template_name": "bitbucket",`

3. Open a .md file, and launch preview with [Sublime Text keybindings](https://github.com/timonwong/OmniMarkupPreviewer#key-bindings)
